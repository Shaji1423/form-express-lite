# 🚀 Form Express Lite

A modern and responsive Form Management System frontend built with **React.js** and **Tailwind CSS**. The application provides separate interfaces for **Users** and **Admins**, allowing seamless form creation, management, and submission through a clean and intuitive UI.

![React](https://img.shields.io/badge/React-19-blue?logo=react)

![TypeScript](https://img.shields.io/badge/TypeScript-Ready-3178C6?logo=typescript)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Shaji1423/form-express-lite.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
