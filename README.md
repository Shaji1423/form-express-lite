# 🚀 Form Express Lite

A modern and responsive form management web application built using React and Tailwind CSS. It provides an intuitive interface for creating, managing, and submitting forms with a clean and user-friendly design.

## 🌟 Features

- 🎨 Modern & Responsive UI
- 📱 Mobile-Friendly Design
- ⚡ Fast Performance
- 📝 Dynamic Form Management
- 🔒 Secure Authentication (if enabled)
- 👨‍💼 Separate User & Admin Roles
- 🌙 Clean and Minimal Interface

## 🛠️ Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- Vite

### Tools
- Git
- GitHub
- Lovable AI

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Shaji1423/form-express-lite.git
```

Navigate into the project:

```bash
cd form-express-lite
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
